# DL_Digits_recognition-ANNcw
# 🧠 Handwritten Digit Recognition using ANN

A Deep Learning project that recognizes handwritten digits using an Artificial Neural Network (ANN).
This application is built with TensorFlow/Keras and deployed using Streamlit.

---

## 📌 Project Overview

This project uses the MNIST dataset to train an ANN model capable of predicting handwritten digits from 0–9.

The application provides:

* Digit prediction using ANN
* Interactive Streamlit interface
* Deep Learning model integration
* Real-time predictions

---

## 🚀 Features

* Handwritten digit classification
* ANN model using TensorFlow/Keras
* Streamlit web application
* Clean and simple UI
* Deployment-ready structure
